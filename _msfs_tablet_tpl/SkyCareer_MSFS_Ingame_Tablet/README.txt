SkyCareer MSFS Ingame Tablet

This package adds an in-game web tablet for SkyCareer in MSFS 2020/2024.

What you get
- Toolbar tablet panel (works in every aircraft)
- 3D tablet HTML template files for cockpit integration by aircraft modders

Install (MSFS 2020/2024)
1. Extract this package folder SkyCareer_MSFS_Ingame_Tablet.
2. Copy the folder to your Community folder.
3. Start MSFS and open the toolbar icon SkyCareer.

Community path examples
- Steam (MSFS 2020): %APPDATA%\\Microsoft Flight Simulator\\Packages\\Community
- MS Store (MSFS 2020): %LOCALAPPDATA%\\Packages\\Microsoft.FlightSimulator_8wekyb3d8bbwe\\LocalCache\\Packages\\Community
- MSFS 2024 (Store): %LOCALAPPDATA%\\Packages\\Microsoft.Limitless_8wekyb3d8bbwe\\LocalCache\\Packages\\Community

3D tablet integration
Template is in html_ui/SkyCareerTablet3D/.
Aircraft creators can map this HTML as a VCockpit instrument onto a tablet mesh.
