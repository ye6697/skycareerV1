# SkyCareerV1 – Vollständiges Benutzer- und Systemhandbuch

Stand: 2026-05-04

## 1. Überblick
SkyCareerV1 ist eine umfassende Airline-Karriere-Simulation mit Vertragsmanagement, Flugbetrieb, Flottenmanagement, Finanzen, Achievements, Leaderboard, Type Ratings und Integrationen für Flugsimulatoren (u. a. MSFS/X-Plane).

## 2. Hauptmodule
- Dashboard und Unternehmensstatus
- Verträge und Einsatzplanung
- Aktive Flüge, Tracking und Auswertung
- Flotte, Wartung und 3D-Hangar
- Finanzen, Credits, Shop und Abos
- Pilotprofil, Konto und Einstellungen
- Achievements, Leaderboard und Type Ratings
- Admin-Bereiche (Discounts, Bildverwaltung, Game Settings)

## 6. Typische End-to-End Workflows
### 6.1 Vertragsflug
1. Vertrag auswählen
2. Flug vorbereiten (Route, Treibstoff, Wetter, ggf. SimBrief)
3. Flug starten und live verfolgen
4. Landung analysieren (Runway Accuracy, Landing Score)
5. Abschluss, Auszahlung, Reputation und Achievements

### 6.2 Flottenwachstum
1. Finanzstatus prüfen
2. Aircraft erwerben/zuweisen
3. Wartung & Versicherungsrisiko managen
4. Höherwertige Verträge annehmen

### 6.3 Skill-Fortschritt
1. Type Ratings starten
2. Missionsziele erfüllen
3. Achievements freischalten
4. Leaderboard-Rang verbessern

- React + Vite Frontend
- React Router mit dynamischem Seitenregister
- React Query für Serverstate
- Base44 SDK/API-Anbindung
- UI-Bausteine über Radix/Shadcn-Komponenten
